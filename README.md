## Run, Build & Deploy

To start up the local server, run
```bash
npm start
```
Open a browser and hit [http://localhost:3000](http://localhost:3000)

<br/>

Building the dist version of the project
```bash
npm run build
```

To deploy the slideshow to [surge](https://surge.sh/), run 
```bash
npm run deploy
```
